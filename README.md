# Student-management-system-

C++ OOP File Handling

📌 Project Overview
The Student Management System is a menu-driven console application developed in C++ that efficiently manages student records using Object-Oriented Programming (OOP) and binary file handling.

It allows users to perform operations like adding, searching, updating, and deleting student data in a structured and efficient way.

🚀 Features
✨ Add Student Record 📋 Display All Students 🔍 Search Student by Roll Number ✏️ Update Student Details ❌ Delete Student Record 🏆 Show Class Topper

🛠 Technologies Used
💻 C++
🧠 Object-Oriented Programming (OOP)
💾 Binary File Handling
